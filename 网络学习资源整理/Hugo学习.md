# Hugo 学习

### 安装

> https://github.com/gohugoio/hugo/releases

### 快速开始

**`hugo new site <projectname>`** - 创建一个 Hugo 主题项目

**项目目录结构**

```bash
├── archetypes
│   └── default.md
├── config.toml
├── content
├── data
├── layouts            
├── static                // 一般存放 css、js、img 等静态资源
└── themes           // 存放主题目录
6 directories, 2 files
```



**`hugo new about.md`** - 会在 `content/` 目录新建 about.md 文件，文件内有自动生成的头

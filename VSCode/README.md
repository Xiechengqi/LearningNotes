## 快捷键

| 键位 | 解释 |
| :---: |:-: |
| **`Ctrl p`** | 搜索文件 |
| **`Ctrk Shift k`** | 删除行 |
| **`Ctrl /`** | 注释行 |
| **`Ctrl Enter`** | 直接跳到下一行 |
| **`Ctrl Shift p`** | 命令面板 |
| **`Ctrl Shift x`** | 扩展面板 |
